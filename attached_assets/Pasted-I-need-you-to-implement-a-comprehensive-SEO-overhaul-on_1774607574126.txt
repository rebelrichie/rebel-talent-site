## GLOBAL CHANGES (apply to every page)

### 1. Add Canonical Tags
Every page is missing a canonical tag. Add a self-referencing canonical to the <head> of every page:
<link rel="canonical" href="https://rebeltalentsystems.com/[PAGE-PATH]" />
Use the correct path for each page (e.g., "/" for homepage, "/services" for services, etc.)

### 2. Add Twitter/X Card Meta Tags
Add these to the <head> of every page (globally, can pull values from og: tags):
<meta name="twitter:card" content="summary_large_image" />
<meta name="twitter:title" content="[same as og:title]" />
<meta name="twitter:description" content="[same as og:description]" />
<meta name="twitter:image" content="https://rebeltalentsystems.com/og-image.png" />

### 3. Fix og:image to Absolute URL
The og:image is currently a relative path (/og-image.png). Change it to the full absolute URL on every page:
<meta property="og:image" content="https://rebeltalentsystems.com/og-image.png" />

### 4. Add Performance Hints to <head>
Add these preconnect/dns-prefetch hints globally in the site <head>:
<link rel="preconnect" href="https://calendly.com" />
<link rel="dns-prefetch" href="https://www.linkedin.com" />
